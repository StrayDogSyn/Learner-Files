About This Project

The Knucklebones Dice Simulator was created for tabletop gaming enthusiasts. Features include:

  • Multiple dice types (d3, d4, d6, d8, d10, d12, d20, d100)
  • Visual dice roll animations
  • Results tracking and history
  • Ability to roll multiple dice simultaneously
